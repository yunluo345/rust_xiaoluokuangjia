# 随机字符串生成器使用示例

## 基本用法

```rust
use gongju::jichugongju::{shengchengsuijizifuchuan, Suijipeizhi, Zifuji};

// 1. 使用默认配置（小写+大写+数字，长度16，完全随机）
let peizhi = Suijipeizhi::default();
let suiji = shengchengsuijizifuchuan(&peizhi);
println!("默认: {}", suiji);  // 例如: aB3xK9mP2nQ7rS4t

// 2. 只生成小写字母
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Xiaoxie])
    .shezhi_changdu(10);
let suiji = shengchengsuijizifuchuan(&peizhi);
println!("小写: {}", suiji);  // 例如: abcdefghij

// 3. 只生成数字
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Shuzi])
    .shezhi_changdu(6);
let suiji = shengchengsuijizifuchuan(&peizhi);
println!("数字: {}", suiji);  // 例如: 123456

// 4. 只生成特殊符号
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Teshu])
    .shezhi_changdu(8);
let suiji = shengchengsuijizifuchuan(&peizhi);
println!("特殊: {}", suiji);  // 例如: !@#$%^&*

// 5. 小写+数字混合
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Xiaoxie, Zifuji::Shuzi])
    .shezhi_changdu(12);
let suiji = shengchengsuijizifuchuan(&peizhi);
println!("小写+数字: {}", suiji);  // 例如: a3b7c2d9e1f4

// 6. 大写+小写+数字+特殊符号（完全随机）
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Daxie, Zifuji::Xiaoxie, Zifuji::Shuzi, Zifuji::Teshu])
    .shezhi_changdu(20)
    .shezhi_hunluan(true);
let suiji = shengchengsuijizifuchuan(&peizhi);
println!("全混合: {}", suiji);  // 例如: aB3!xK9@mP2#nQ7$rS4%

// 7. 分组模式（按顺序，不混乱）
// 前4位大写，中间6位数字，后4位小写
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Daxie, Zifuji::Shuzi, Zifuji::Xiaoxie])
    .shezhi_hunluan(false)
    .shezhi_fenzuchangdu(vec![4, 6, 4]);
let suiji = shengchengsuijizifuchuan(&peizhi);
println!("分组: {}", suiji);  // 例如: ABCD123456efgh

// 8. 批量生成（生成10个随机字符串）
use gongju::jichugongju::piliang_shengcheng;

let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Xiaoxie, Zifuji::Shuzi])
    .shezhi_changdu(8);
let liebie = piliang_shengcheng(&peizhi, 10);
for (i, s) in liebie.iter().enumerate() {
    println!("第{}个: {}", i + 1, s);
}
```

## 高级用法

### 生成强密码
```rust
// 16位强密码：大写+小写+数字+特殊符号
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![
        Zifuji::Daxie,
        Zifuji::Xiaoxie,
        Zifuji::Shuzi,
        Zifuji::Teshu
    ])
    .shezhi_changdu(16)
    .shezhi_hunluan(true);
let mima = shengchengsuijizifuchuan(&peizhi);
```

### 生成验证码
```rust
// 6位数字验证码
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Shuzi])
    .shezhi_changdu(6);
let yanzhengma = shengchengsuijizifuchuan(&peizhi);
```

### 生成用户名
```rust
// 8位小写字母+数字
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Xiaoxie, Zifuji::Shuzi])
    .shezhi_changdu(8);
let yonghuming = shengchengsuijizifuchuan(&peizhi);
```

### 生成订单号
```rust
// 格式：4位大写字母 + 8位数字 + 4位小写字母
let peizhi = Suijipeizhi::xin()
    .shezhi_zifuji(vec![Zifuji::Daxie, Zifuji::Shuzi, Zifuji::Xiaoxie])
    .shezhi_hunluan(false)
    .shezhi_fenzuchangdu(vec![4, 8, 4]);
let dingdanhao = shengchengsuijizifuchuan(&peizhi);
```

## API 说明

### Zifuji 枚举
- `Zifuji::Xiaoxie` - 小写字母 (a-z)
- `Zifuji::Daxie` - 大写字母 (A-Z)
- `Zifuji::Shuzi` - 数字 (0-9)
- `Zifuji::Teshu` - 特殊符号 (!@#$%^&*()_+-=[]{}|;:,.<>?)

### Suijipeizhi 配置
- `zifuji: Vec<Zifuji>` - 使用的字符集列表
- `changdu: usize` - 生成字符串的总长度
- `hunluan: bool` - 是否完全随机混乱（true=完全随机，false=按分组）
- `fenzuchangdu: Option<Vec<usize>>` - 每组的长度（仅在 hunluan=false 时有效）

### 方法
- `Suijipeizhi::xin()` - 创建新配置（使用默认值）
- `shezhi_zifuji(vec)` - 设置字符集
- `shezhi_changdu(n)` - 设置长度
- `shezhi_hunluan(bool)` - 设置是否混乱
- `shezhi_fenzuchangdu(vec)` - 设置分组长度
- `shengchengsuijizifuchuan(&peizhi)` - 生成单个随机字符串
- `piliang_shengcheng(&peizhi, n)` - 批量生成 n 个随机字符串

## 性能特点

1. **高性能**：使用 `rand` crate 的高效随机数生成器
2. **零拷贝**：字符集使用静态字符串，无额外分配
3. **预分配**：结果字符串预分配容量，避免多次扩容
4. **Fisher-Yates 洗牌**：使用标准算法打乱字符顺序，O(n) 时间复杂度
5. **链式调用**：配置使用 Builder 模式，代码简洁优雅
